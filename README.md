## Description

cub3D is a first-person ray-casting maze renderer in C, using miniLibX. The program reads a `.cub` scene file (textures, floor/ceiling colors, and map), validates it, and renders a 3D view the player can move through with WASD and arrow keys.

## Instructions

```bash
make
./cub3D maps/valid/simple.cub
```

- One argument: path to a `.cub` file.
- Close the window with ESC or the red cross.
- `make clean`, `make fclean`, and `make re` are available.
